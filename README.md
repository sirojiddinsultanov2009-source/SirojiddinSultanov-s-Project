# E-Commerce Product Catalog

Plantify is a responsive product catalog built with HTML, CSS and JavaScript. Products are loaded from the Jsonbek API with the Fetch API.

## Features

- Load products from `https://jsonbek.uz/api/products`
- Loading and error states
- Product cards with image, name, price, rating and category
- Like products with LocalStorage persistence
- Add products to cart with quantity support
- Cart page with increase, decrease, remove and total price
- Responsive layout for desktop, tablet and mobile

## Technologies

- HTML
- CSS
- JavaScript ES6+
- Fetch API
- LocalStorage

## Run

Open `index.html` in a browser or serve the folder with a local server.
